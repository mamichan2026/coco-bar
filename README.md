# coco-bar
coco bar (Visual Guidance Bar) のフィードバック・不具合報告用スペースです。ご意見・ご要望をお気軽にどうぞ。
This repository is for feedback and bug reports about coco bar (Visual Guidance Bar).
Feel free to post any comments, issues, or suggestions in the Issues tab.

プライバシーポリシー / Privacy Policy

coco bar (Visual Guidance Bar) は、お客様の個人情報を収集・保存・送信しません。
音声認識機能はすべて端末内(ローカル)で処理され、外部に音声データが送信されることはありません。

本アプリのフィードバック・不具合報告にはGitHubを利用しています。
GitHubの利用に伴い収集される情報については、GitHub自身のプライバシーポリシーが適用されます。
詳しくは GitHub Privacy Statement（https://docs.github.com/site-policy/privacy-policies/github-general-privacy-statement）をご覧ください。

---

coco bar (Visual Guidance Bar) does not collect, store, or transmit any personal information.
All speech recognition processing is performed locally on your device, and no audio data is sent externally.

This app uses GitHub for feedback and bug reports.
Information collected through GitHub is subject to GitHub's own Privacy Statement.
For details, see the GitHub Privacy Statement (https://docs.github.com/site-policy/privacy-policies/github-general-privacy-statement).
